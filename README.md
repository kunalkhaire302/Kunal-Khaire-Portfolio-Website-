# Kunal Khaire Portfolio

A modern, interactive personal portfolio website for Kunal Khaire. This site showcases technical skills, projects, education, experience, and contact information, featuring a professional profile photo and visually engaging design.

## Features

- Responsive design (mobile/tablet/desktop)
- Animated hero section with profile photo
- Skills and experience cards with glassmorphism effects
- Interactive project gallery with GitHub/demo links
- Contact form and social media badges
- Smooth scroll and fade-in animations

## Usage

Open `index.html` in your browser to view the portfolio.  
All source files are included in this project directory for customization.

## Built With

- HTML5, CSS3, JavaScript
- CSS Grid & Flexbox
- Font Awesome icons
